Test Connection

How to use

  • Enter the AUTH_PASSWORD you set as environment variable in your Cloudflare Worker.
  • Click "Authenticate". A token will be stored in your browser.
  • Then use the "Test Connection" button to verify your D1 database is reachable.
  • The same token will be used by the extractor to save extraction results automatically.
